Basic Sketch Data Collection Interface - Instructions

About

This interface is a basic sketch data collection interface for collecting sketches with minimalist properties. The primary purpose of this interface is for usage in academic code, such as for class projects and thesis/dissertation code. The interface provides canvas resizing, sketch labeling, and collection downloading as your core needs.

Usage

  • Resize: Enter the width and height in pixels, and then click the resize button to automatically resize the canvas to those dimensions.
  • Label: Enter the name of the sketch as the label.
  • Clear: Click the clear button to clear the entire canvas of strokes.
  • Undo: Click the undo button to remove the most recent stroke drawn.
  • Save: Click the save button to store the active sketch into the active collection.
  • Download: Click the download button to access all the saved sketches as a download link and to reset the collection for any subsequent sketch data collection.

Data Structure

The data structure from the data collection is very minimalist, and is broken down into four levels.
  • Sketch Collection: The top level is the sketch collection, which consists of an array of sketch objects.
  • Sketch: The sketch object contains the following: canvasWidth, canvasHeight, interpretation, and array of stroke objects.
  • Stroke: The stroke object contains the following: array of point objects.
  • Point: The point object contains the following: x, y, time.
