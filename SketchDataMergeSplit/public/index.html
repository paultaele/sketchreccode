Instructions

About

The purpose of this code is to work with multiple sketch data files, and merge or split the sketches into new files.

Guide

  1. Loading Data: Use the "Choose Files" button at the top to select one or more sketch data files. This will load the labels of the sketches from each of those files on the left side.
  2. Moving Data: Use the "Move" buttons in the middle to transfer the labels to the left and right listboxes.
  3. Sorting Data: Use the "Sort" buttons to sort the labels in the target listbox in alphabetical order.
  4. Downloading Data: Use the "Download" to download the labels' sketches in the right listbox into a new data file.
